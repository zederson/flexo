
# RELE
this project supported 4 reles
## start
```
RELE${index},${1|0}
```
### on rele 1
```
RELE1,1
```

### off rele 1
```
RELE1,0
```

# MOTOR STEP
## set speed
```
STEP_SPEED|50
```

## set steps
```
STEP_ON|30
```

# ULTRASONIC
## print value in CM
```
ULTRASONIC[293.43]
```

# SENSOR RAIN
## RAIN ?
$(true | false)

```
RAIN[true]
```

## INTENSITY
## low, moderate or hard
$(low | moderate | hard)

```
RAIN_INTENSITY[low]
```
